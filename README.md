# CalcuPortal - Free Online Calculator Website

CalcuPortal is a GitHub Pages friendly static calculator website. It includes multiple calculator categories, individual calculator pages, SEO content sections, FAQs, sitemap, robots.txt, and basic legal pages.

## Website Structure

- Homepage
- Calculator Categories Page
- L2 Category Pages:
  - Math Calculators
  - Finance Calculators
  - Health & Fitness Calculators
  - Date & Age Calculators
  - Conversion Calculators
  - Games & Random Calculators

- L3 Calculator Pages:
  - H1 Title
  - Calculator Tool Section
  - SEO Content Section
  - FAQs Section

- Important Pages:
  - About Us
  - Contact Us
  - Privacy Policy
  - Disclaimer
  - Advertise

- SEO Files:
  - sitemap.xml
  - robots.txt

## How to Run Locally

1. Download or clone this repository.
2. Open the `index.html` file in your browser.
3. The website will run locally without any server setup.

## GitHub Pages Setup

1. Upload all files to your GitHub repository root.
2. Go to repository **Settings**.
3. Click **Pages**.
4. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Click **Save**.
6. Open your GitHub Pages URL after deployment.
