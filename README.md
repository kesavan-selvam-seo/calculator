# CalcStack - Calculator Category Website

This is a GitHub Pages friendly static calculator website.

## Structure
- Homepage
- Categories page
- L2 category pages: Generic, Finance, Health, Student, Sports
- L3 calculator pages: H1 + Calculator Tool + SEO Content + FAQs
- About, Contact, Privacy Policy, Disclaimer
- sitemap.xml and robots.txt

## How to run
Open `index.html` in your browser.

## GitHub Pages
Upload all files to your repository root and enable GitHub Pages from repository settings.

## Before live
Replace `example.com` in canonical, sitemap and robots with your own domain.
